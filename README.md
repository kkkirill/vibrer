# Vibrer

## Online audio distribution platform and music sharing website

Requirements:
* python 3.7.4
* pipenv 2018.11.26

Install pipenv package:
```bash
    pip3 install pipenv==2018.11.26
```

Installation dependencies:
```bash
    cd vibrer
    # (you should get to the same level with Pipfile)
```
```bash
    pipenv --python 3.7.4
    pipenv install --dev
```

Activate pipenv:
```bash
    pipenv shell
```

Deactivate pipenv:
```bash
    exit
```
